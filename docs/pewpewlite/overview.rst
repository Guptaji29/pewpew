PewPew Lite FeatherWing
***********************


PewPew Lite is a FeatherWing that you can use with any of the Adafruit Feather
development boards. It's exactly the size of all FeatherWings, with a green-red
display 2cm on a side, six clicky buttons and a switch for an optional battery.
